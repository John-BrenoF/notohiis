# Como Contribuir

1. **Abra uma Issue:** Crie uma issue detalhada e descritiva sobre o problema ou adição. A proposta deve seguir obrigatoriamente a filosofia do editor.
2. **Crie a Branch:** Use o formato `<tipo>/<nome-descritivo>`. Exemplo: `add/suporte-utf8` ou `correcao/bug-scroll`.
3. **Faça a Modificação e Comente:** Toda área de código modificada ou adicionada deve conter um comentário explicativo (preferencialmente em Português, Inglês aceito) iniciando com uma tag obrigatória:
   - `[Correção]`: Resolução de bugs e erros.
   - `[Pedido]`: Novas funcionalidades.
   - `[Melhoria]`: Otimizações de código existente.
   - `[Refatoração]`: Reescrita de estrutura sem alterar comportamento.
   - `[outro]`: Código que não se encaixa em nenhuma categoria.
   - `[add]`: Adicionou algo novo que não é necessariamente uma funcionalidade.
   - `[updade_plugin]`: Melhoria relacionada a plugin.
   Exemplo: `# [Correção] Resolve travamento ao fechar o editor`
4. **Envie o Pull Request:** Submeta o código e abra o PR apontando para a branch principal.
